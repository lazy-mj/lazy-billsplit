# ✂️ 전화요금 계정분할 자동화 도구(Lazy-billsplit)

부서별로 청구되는 전화요금을 **계정번호(과제번호)별로 자동 분할**해주는 웹 도구입니다.
설치·로그인 없이 브라우저에서 바로 쓸 수 있고, 서버 없이 전부 내 PC 안에서 처리됩니다.

**🔗 바로 사용하기: [lazy-mj.github.io/lazy-billsplit](https://lazy-mj.github.io/lazy-billsplit/)**

---

## ✨ 주요 기능

-  **부서번호 + 전화번호 기준 자동 매칭**으로, 매달 배포되는 원본 전화요금 파일을 업로드하면 등록해둔 계정번호별로 자동 합산돼요
-  **업로드한 파일은 서버나 페이지에 저장되지 않고, 오직 방문한 내 PC(브라우저) 안에서만 처리**되어 안전해요 — 개인정보·요금 데이터가 외부로 전송될 일이 없습니다
-  **분할된 표를 바로 복사**해서 한글(HWP)·워드 기안문에 표 형태 그대로 붙여넣을 수 있어요
-  **누락 검증 기능**으로 오류를 줄였습니다
  - 계정 매핑에 등록되지 않은 번호(신입사원 등)를 별도로 잡아서 보여줘요
  - 원본 파일 안의 부서별 공식 합계와 우리 분할 결과를 서로 대조해서, 실제로 새는 금액이 없는지 이중으로 확인해요
-  **계정 매핑은 브라우저에 자동 저장**되어 다음 달에도 다시 입력할 필요가 없어요 (원치 않으면 저장 끄기도 가능)
-  결과는 **엑셀 파일로도 다운로드** 가능 (부서별 셀 병합, 서식 포함)
-  표 검색·정렬, 드래그 앤 드롭 업로드 등 실무에서 바로 쓰기 편하게 구성했습니다

## 📝 사용법 (요약)

1. 계정 매핑 — "계정 매핑 양식 다운로드"로 엑셀을 받아 부서원별 계정번호를 채운 뒤 업로드
2. 부서별 전화사용 내역 업로드 — 연구지원실(시설관리실) 배포 원본 파일을 그대로 업로드
3. 분할 실행 — 계정번호별 합산 결과 확인, 표 복사 또는 엑셀 다운로드
4. 검증 — 부서별 합계 일치 여부와 매핑 누락 항목을 자동으로 확인

> ✔️ 자세한 사용법은 페이지 안의 **"사용법 펼쳐보기"**를 참고해주세요.

## 🔒 보안 관련

- 네트워크 요청 코드가 전혀 없고, `Content-Security-Policy`로 외부 통신을 브라우저 단에서 차단해뒀습니다
- 계정 매핑은 브라우저 로컬 저장소(`localStorage`)에만 저장되며, 그 외 어떤 값도 서버로 전송되지 않습니다
- 다만 정적 페이지 특성상 소스 코드(계산 로직)는 누구나 열람 가능합니다 — 데이터만 비공개이며 코드 자체는 공개된 도구입니다

## 🛠 기술 스택

순수 HTML / CSS / JavaScript + [SheetJS(xlsx)](https://sheetjs.com/)로 만든 정적 웹앱입니다. 별도 서버·빌드 과정 없이 GitHub Pages로 바로 배포됩니다.

```
index.html   메인 페이지
style.css    스타일
app.js       엑셀 파싱 · 분할 계산 · 검증 로직
assets/      캐릭터 이미지 등
js/          xlsx 라이브러리
```
