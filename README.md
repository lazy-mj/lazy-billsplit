# 전화요금 계정분할 자동화
   ![전화요금 계정분할 자동화](./assets/readme_thumbnail.png)
> **사내 업무지원 도구**\
> 전화요금 명세서를 계정별로 자동 분할하는 브라우저 기반 웹 애플리케이션

## 🌐 바로 사용하기

**GitHub Pages** 👉 https://lazy-mj.github.io/lazy-billsplit/

별도의 설치 없이 브라우저에서 바로 사용할 수 있습니다.

## ✨ 주요 기능

-   🌟 분할 결과 복사해서 바로 기안문에 붙이기
-   🔍 번호별요금 시트 자동 인식
-   📅 월 정보 자동 추출
-   📊 계정별 전화요금 자동 분할
-   ⚠️ 누락 데이터 확인
-   📥 결과 엑셀 다운로드/바로 복사
-   💾 계정 매핑내용 (localStorage) 저장

## 🔒 개인정보 및 보안

이 프로그램은 **모든 작업을 사용자의 브라우저에서만 처리**합니다.

-   업로드한 엑셀 파일은 서버로 전송되지 않습니다.
-   GitHub에는 사용자 데이터가 저장되지 않습니다.
-   모든 분석은 브라우저 메모리에서 수행됩니다.
-   결과 파일은 사용자의 PC로 직접 다운로드됩니다.

## 🚀 사용 방법
![사용 예시](./assets/readme_demo.gif)
1.  자동화봇(계정 매핑) 엑셀 업로드
2.  전화요금 엑셀 업로드
3.  자동 분석 실행
4.  결과 엑셀 다운로드 or 복사해서 바로쓰기
