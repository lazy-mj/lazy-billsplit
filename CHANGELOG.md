# CHANGELOG

이 문서는 "전화요금 계정분할 자동화" 도구의 주요 변경 이력을 담습니다.
사내 소규모 도구 특성상 엄격한 시맨틱 버저닝 대신, 굵직한 변경 단위로만 기록합니다.

## v3 (현재 배포 버전 · 2026-07-15)

### 추가
- 계정 매핑을 파일 업로드 방식으로 전환, "계정 매핑 양식 다운로드" 버튼으로 표준 엑셀 양식 제공
- 계정 매핑 검색(이름·전화번호·부서번호), 결과 표 헤더 클릭 정렬
- 결과 표 **"📋 표 복사하기"** — 한글(HWP)·워드 등에 실제 표 형태로 바로 붙여넣기 가능
- **"④ 원본 부서별 합계와 교차 검증"** — 원본 파일 속 부서별 소계와 우리 분할 합계를 자동 대조해 누락을 이중으로 확인
- **"⑤ 계정 매핑이 안 된 항목"** — 등록 누락 의심 항목을 부서 단위로 표시
- 종비용명 항목을 사용자가 직접 수정 가능하도록 변경 (기존엔 "공공요금" 고정)
- 적요(비고) 월 자동 인식: 파일명의 청구월(예: "6월(5월 사용분)")을 우선 인식하도록 개선
- 드래그 앤 드롭 업로드, 진행상태 표시, Toast 알림, 계정 매핑 로컬 저장 여부 선택(체크박스)
- 공공기관 업무시스템 톤의 UI 리디자인 (STEP 인디케이터, 카드형 레이아웃)
- 캐릭터 일러스트 적용
- GitHub Pages 정적 배포 지원 (외부 CDN 제거, `Content-Security-Policy` 적용)

### 수정
- 계정 매핑 검색창 아이콘·텍스트 겹침 현상 수정
- 결과 카드 하단 버튼과 안내 문구 사이 간격 조정
- **[버그 수정]** 부서별 합계 검증 로직에서, 등록 누락된 금액까지 "우리 계산 합계"에 포함시켜 항상 "일치"로 표시되던 문제 수정 → 이제 실제로 계정에 분할된 금액만 반영해 누락을 정확히 잡아냄
- 표 복사 시 헤더가 빠지거나 "표 안의 표"로 잘못 인식되던 문제 수정 (단순한 `<table>` 구조로 재작성)

## v2 (첫 웹 버전)

- 기존 엑셀 매크로(VBA)의 계산 로직을 웹으로 이식 (부서번호+전화번호 매칭 → 계정번호별 합산)
- 매칭 안 된 항목을 조용히 누락시키던 매크로의 동작을 개선해, 화면에 별도로 표시하도록 변경
- 브라우저 로컬 저장(localStorage)으로 계정 매핑 유지
